# Static 静态资源目录

这个目录包含 GTPlanner 的静态资源文件，主要用于测试和演示。

## 文件说明

### test_sse_chat.html

完整的 SSE 聊天测试界面，提供以下功能：

- **实时聊天界面**: 美观的聊天界面，支持用户和助手的消息显示
- **SSE 流式响应**: 实时接收和显示流式响应数据
- **工具调用监控**: 实时显示工具调用状态和进度
- **多语言支持**: 支持中文、英文、日文、西班牙文、法文
- **状态面板**: 显示连接状态、会话信息、事件日志等
- **配置选项**: 可配置语言、元数据包含等选项

#### 主要特性

1. **响应式设计**: 适配桌面和移动设备
2. **实时状态更新**: 显示连接状态、消息计数、工具调用等
3. **事件日志**: 记录所有 SSE 事件的详细日志
4. **错误处理**: 优雅的错误处理和用户提示
5. **美观界面**: 现代化的 UI 设计，良好的用户体验

#### 使用方法

1. 启动 GTPlanner 服务器：
   ```bash
   python start_server.py
   ```

2. 打开浏览器访问：
   ```
   http://localhost:11211
   ```

3. 在输入框中输入您的需求，点击发送按钮开始对话

#### 技术实现

- **纯 HTML/CSS/JavaScript**: 无外部依赖，轻量级实现
- **Fetch API**: 使用现代浏览器的 Fetch API 发送请求
- **Stream Reader**: 使用 ReadableStream 读取 SSE 数据流
- **事件驱动**: 基于事件的响应处理机制

## 部署注意事项

1. **生产环境**: 在生产环境中，建议使用专业的静态文件服务器（如 Nginx）来提供静态文件服务
2. **缓存策略**: 可以为静态文件设置适当的缓存策略
3. **安全考虑**: 确保静态文件不会暴露敏感信息
4. **性能优化**: 可以考虑压缩静态文件以提高加载速度
